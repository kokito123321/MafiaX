# Mafia X - აპლიკაციის ტესტირების გზები

## 1. Replit Development Server (ყველაზე მარტივი)

Replit-ში არის უკვე კონფიგურირებული ports:
- **აპლიკაციის ვებვერსია:** http://localhost:8081
- **API სერვერი:** http://localhost:8080
- **Expo დეველოპერ სერვერი:** http://localhost:3000

**გამოსაყენებლად:**
1. გახსენით Replit-ის პროექტი
2. დააჭირეთ "Run" ღილაკს
3. გახსენით http://localhost:8081 ბრაუზერში

## 2. Expo Go ალტერნატივები

### Expo Development Build
```bash
# თუ გინდათ ააშენოთ სპეციალური build
npx expo install --fix
npx expo run:android
npx expo run:ios
```

### Expo Web (ბრაუზერში)
```bash
npx expo start --web
```

## 3. მობილურის ტესტირება უფასო ხელსაწყოებით

### Appetize.io (უფასო ვერსია)
1. ატვირთეთ Expo ბილდი
2. გამოიყენეთ https://appetize.io ტესტირებისთვის

### Android Studio Emulator
1. დააყენეთ Android Studio
2. შექმენით ვირტუალური მოწყობილობა
3. გაუშვით: `npx expo run:android`

## 5. ცნობილი პრობლემები და ამჟამინდელი სტატუსი

### ✅ გასწორებულია:
- CORS უსაფრთხოების პრობლემა
- ავატარის ზომის ლიმიტი (500KB → 100KB)

- LiveKit ვიდეო შეიძლება არ მუშაობდეს localhost-ზე

## 6. რეკომენდაციები

1. **დაიწყეთ Replit-ით** - ყველაზე მარტივია
2. **გამოიყენეთ Chrome** - უკეთესია ვებვერსიისთვის
3. **ტესტირეთ მობილურზე** - Android Studio Emulator-ით
4. **შეამოწმეთ API** - Postman-ით ან curl-ით

## 7. API ენდპოინტების ტესტირება

```bash
# ჯანმრთელობის შემოწმება
curl http://localhost:8080/api/health

# რეგისტრაცია
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"123456","name":"TestUser"}'

# ლოგინი
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"123456"}'
```
